Front 개발 블로그(4)

저번에 말했던대로 이번에는 github에 git 을 연결을 해보려고 한다
우선 지금까지의 작업본을 commit 을 하자

나는 보통 vscode 에서 기본으로 제공해주는 git 이나 sourcetree, 터미널 3가지를 같이 사용한다
merge 는 터미널, commit 은 vscode, tree 를 보거나 roll back 같은 작업 은 sourcetree 를 선호한다

위 사진처럼 스테이징 한후 엔터를 눌러주면 git 에 커밋이 완료 된다
그리고
github에 가서 repository 를 생성해주면

아래와 같은 화면이 나올거다

이제 쓰여있는 대로 git remote add orgin ~~ 명령어를 입력하면 되는데
git 이 위치한 /savemoney 로 가서 헤당명령어를 입력한후 origin master 에 push 를 하면

에러가 나올거다

github 에 ssh key 를 등록을 하지않아서 나오는 오류이다
예전에는 username, email, password 를 입력하면 됐지만 이제
ssh key를 등록해줘야 할수있게 바뀌였다고 들었다

ssh key 를 등록하는 방법은
https://gastroad.github.io/2020/11/12/git-SSH-key/
에 정리해 두었다

ssh key 를 git hub 에 등록후 push 에 성공했다면

아래 화면 처럼 나오게 될것이다
여기서 contributors 가 2명인이유는 앞에서도 말했듯이
vscode, sourcetree, 터미널 3가지를 동시에 사용하기때문에 commit 을 하는 계정이 2개라 contributors 가 2명으로 나타난다
