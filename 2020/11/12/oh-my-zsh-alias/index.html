oh-my-zsh alias 설정

오늘은 alias 설정에 대해서 포스팅을 해보려고한다
일단 zsh 에서 alias 설정하는 방법은 간단하다
우선 터미널에 접속을해서

커맨드를 입력한다 그러면

아래 사진처럼 .zshrc 라는 파일이 있을것이다

를 입력하면 .zshrc 파일이 열릴 것이다

형태로 작성후 저장, 재실행 후 단축어만 입력하면 명령어가 실행이 된다
테스트로 hello 를 입력하면 echo alias-test 가 실행되게 한번 해보겠다

.zshrc 파일을 vi 로 연후 insert 버튼을 누르면 파일 편집이 가능하다

.zshrc 파일안에 위 사진처럼 명령어를 입력후 ESC 키 [:] [w][q] 를 입력하면 저장하고 파일나가기를 할수 있다 이때 한영키에 주의해서 하도록 하자

.zshrc 파일에 alias 를 설정하고 저장한후

명령어를 통해 zsh 를 재실행 해주도록 한다
이제 터미널에 hello 를 입력하면

echo alias-test 가 정상적으로 출력이 되었다.
이제 alias 를 어떻게 활용 하는지에 대해서 알아보도록 하겠다
개발을 할때 개발서버를 키는 부분을 alias 를 통하여 간단하게 만들어 보겠다

savemoney 라는 단축어 를 입력하면 savemoeny 프로젝트로 이동하여 npm start 를 통해 개발서버를 켜주는 명령어이다

위와 같은 규칙으로 새로운 프로젝트를 시작했을때 미리 설정을 해두면
여러 프로젝트를 왔다 갔다 코딩할때 매우 편리하게 사용할수 있다
다만 path 가 변경되면 alias 설정도 다시 해 줘야 한다.
추가로 alias 를 통해 편하게 사용할수 있는 부분이 있으면 계속해서 추가하도록 하겠다.
