git hub 에 SSH key 등록하기

git hub 에 ssh key 를 등록하는 방법에 대해서 알아보도록 하자
mac os 기준으로 SSH key 를 등록할것이니 대부분 비슷하겠지만
mac 이 아니라면 명령어가 조금 다를수도 있다

SSH key 란

서버에 접속 할 때 비밀 번호 대신
사용하는 기능이다
비밀번호 보다 높은 수준의 보안을 필요로 하거나 로그인 없이 자동으로 서버에 접속 할 경우 사용한다
SSH key 는 비대칭 암호방식을 사용한다
public key 와 private key 2가지를 생성한다
public key 는 평문을 암호화 private key 는 암호문을 다시 평문으로 변환한다그래서 public key 를 github 에 업로드 하고
private key 는 로컬에만 두고
대조를 해서 사용자 를 인증을 하는 방식이다
이제 SSH key 를 등록하도록 해보겠다

목차

  1. SSH key 확인
  2. SSH keygen</
  3. SSH key copy
  4. gih hub 등록

1.SSH key 확인

우선 터미널에 들어가 위 명령어를 순서대로 입력한다
만약 SSH key 를 생성한 적이 있으면 이때 공개키가 있는것을 확인할수 있다
공개키가 있으면 2번 단계를 건너뛰고 3번으로 가면 된다.

2. SSH keygen

위 명령어를 입력하면 ssh key 가 생성이 될것이다
-t 는 타입을 지정해주는 옵션이다
-C 는 코맨트 를 입력해주는 옵션이다
-b 는 비트수를 입력하는 옵션이다
그래서 rsa 타입으로 2048 비트 인 SSH key 를 생성해준다는 뜻이다
2048비트와 rsa 타입은 기본 값이므로

으로만 입력해도 github에 등록하는데는 문제가 없다

2. 3. SSH key copy

이제 SSH key 를 생성 했으니 복사 하도록 하겠다

명령어를 입력하면
공개키가 터미널에 나올것이다 그부분을 복사하면 된다

명령어를 통해 바로 클립보드로 복사할수도 있다

4. git hub 등록

ssh public key를 클립보드에 복사했다면
github 에 접속후 우측상단에 프로필을 클릭한다음 setting 에 들어간다

그후SSH and GPG keys 에 들어간뒤 new SSH key 를 클릭한다

Key 부분에 복사한 public key 를 등록하면 SSH key 등록이 된다

이제 내 Repositories 에 있는 Repository 는 전부 연결이 정상작동 할거다
